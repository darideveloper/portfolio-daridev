### Comparativa de Herramientas de Renderizado

#### WeasyPrint vs ReportLab: ¿Cuál Elegir?

Esta es una de las decisiones técnicas más importantes del proyecto:

- **WeasyPrint:** Renderiza HTML y CSS a PDF. Ideal si tu equipo domina frontend. Permite iteraciones rápidas de diseño.
- **ReportLab:** Dibuja el PDF directamente con Python. Ofrece control absoluto y mejor rendimiento en reportes masivos, aunque con mayor curva de aprendizaje.

## Preguntas Frecuentes (FAQ)

### ¿Es difícil mantener un sistema de reportes en Django?
No, si la arquitectura es correcta y desacoplada desde el inicio.

### ¿Puedo usar CSS moderno en los PDFs?
Sí. WeasyPrint soporta Flexbox, Grid y gran parte del estándar CSS actual.

### ¿Qué pasa si los reportes tardan en generarse?
Debe usarse una cola asíncrona y notificar al usuario cuando el PDF esté listo.

### ¿Necesito licencias comerciales?
No. WeasyPrint y ReportLab Open Source tienen licencias permisivas para uso comercial.
