## Project Description  

**Opocosocial Canarias** is an **online learning platform** specifically designed for **Social Work employees** in the Canary Islands.  
The main goal of this first iteration is to provide a **modern, secure, and user-friendly user management module** that allows professionals to access training content and manage their accounts independently.  

The project is developed with **Django**, one of the most robust frameworks for web applications, ensuring a reliable, scalable environment that is ready to grow with the platformâ€™s future needs.  

---  

## **Key Features**  

* **Complete user management:** Registration, login, account activation, and password recovery.  
* **Customizable profile:** Ability to update personal data and upload a profile picture.  
* **Advanced security:** Authentication with **JWT**, password encryption, and data validation.  
* **Intuitive user experience:** Enhanced admin panel with **Jazzmin**, making internal management easier.  
* **Scalability ready:** Architecture designed to incorporate future learning modules, progress tracking, and certifications.  
* **Cloud integration:** Use of **Amazon S3** for secure storage of user images.  

---  

## **Technologies Used**  

### **Backend**  
* **Django 4.2:** Core framework of the application.  
* **Django Rest Framework (DRF):** Creation of a robust API for user management.  
* **SimpleJWT:** Security through access and refresh tokens.  
* **PostgreSQL:** Powerful and reliable database for information management.  

### **Storage & Media**  
* **Amazon S3 + django-storages + boto3:** Cloud-based management of user files and images.  
* **Pillow:** Image processing and validation.  

### **Interface & Administration**  
* **Django Jazzmin:** Customized and user-friendly admin panel.  

### **Testing & Quality Assurance**  
* **Selenium & BeautifulSoup:** Automated tests to ensure quality and performance.  

---  

## **Challenges and Learnings**  

* **Sensitive data security:** Implementation of strong mechanisms to protect passwords and access.  
* **Account recovery:** Efficient and reliable password reset flow.  
* **Modular scalability:** Building a solid foundation to integrate learning modules in future phases.  
* **Admin panel optimization:** Creating an accessible and easy environment for administrators and trainers.  

---  

## **Results**  

The **Opocosocial Canarias** project delivers a **modern and secure digital platform** that meets the training needs of the **Social Work** sector.  
In this first phase, the user system ensures **reliable, secure, and customizable access**, laying the groundwork for future online learning features.  

This solution represents a decisive step toward the **digital transformation of the sector**, giving organizations the ability to manage training and users in a professional, efficient, and scalable way.
